# Car-Price-Prediction
Car Price Prediction using Machine Learning | Developed during a Data Science Internship at AI Variant

## Live Demo
🚀 https://car-price-prediction-tharun.streamlit.app/

## Project Objective
The objective of this project is to predict car prices based on vehicle features using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Streamlit

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Model Training and Evaluation
- Real-Time Car Price Prediction
